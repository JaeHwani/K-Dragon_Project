# 로그인 및 회원가입 페이지

## **Front end**
- **login** : 로그인 페이지
- **join** : 회원가입
- **confirmId** : 아이디 중복 체크
- **checkPwForm** : 정보수정 전 비밀번호 확인
- **userUpdateForm** : 정보수정


## **Back end**
- **checkPw** : 비밀번호 확인
- **joinAction** : 회원가입
- **loginAction** : 로그인
- **logoutAction** : 로그아웃
- **userUpdate** : 정보수정
